# Install machbase-neo

### Download and unarchive file.

Unarchive downloaded file.

```sh
$ unzip machbase-neo-EDITION-{{site.latest_version}}-OS-ARCH.zip
```

### Confirm executable.

```sh
machbase-neo version
```
